# FUTURE_FS_02


**Mini CRM – Client Lead Management System**

**Project Overview**

This project is a **Mini CRM (Client Lead Management System)** built as part of the Full Stack Web Development training program (Task 2).

The application allows users to **manage client leads efficiently** by adding lead details, tracking their status, and organizing them in a simple dashboard interface.

This project demonstrates how businesses track potential clients coming from website contact forms and manage their conversion process.

**Live Website**

https://rishitha-dhanala.github.io/FUTURE_FS_02/

**Features**

• Lead management dashboard with statistics
• Add new client leads
• View leads in a structured table
• Lead status tracking (New / Contacted / Converted)
• Sidebar navigation (Dashboard, Leads, Settings)
• Dark mode / Light mode toggle
• Data stored using browser localStorage
